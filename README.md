# Primeros Algoritmos en Python
Repositorio donde guardo mis primeros ejercicios y algoritmos en Python
## Proyectos incluidos
### Hola
Programa sencillo que solicita tu nombre y edad para mostrarte un mensaje personalizado.

### Calculadora basica
Programa que solicita dos datos (numeros) y la opracion sencilla que desee realizar (+, -, *, /) y muestra el resultado final.

### Calculadora de edad
Programa que solicita tu nombre y edad dandote como resultado un mensaje con tu nombre, tu edad actual, el año aproximado en que naciste y edades futuras.

### Descuento
Programa que solicita un numero y si el numero es mayor a 500 aplica un descuento del 10% y si es menor aplica un descuento del 4%.

### Mayor o menor de edad
Programa que solicita tu nombre y edad y como resultado muestra tus datos indicando si eres un niño, adolescente o adulto junto a un mensaje personalizado.
